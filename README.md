# 天课计算器 v2 · Zakat Calculator

基于哈乃斐学派教法的现代化天课计算工具，采用伊斯兰几何艺术美学设计。

## 设计特点

- **美学风格**: 伊斯兰几何艺术 + 现代极简主义
- **配色方案**: 深青金石蓝 (#1a365d) + 金色点缀 (#d4af37) + 象牙白背景
- **字体组合**: Amiri (阿拉伯文) + Noto Serif SC (中文标题) + Plus Jakarta Sans (界面文字)
- **装饰元素**: 几何图案背景、星形徽章、优雅的渐变效果

## 功能特性

- 🟡 **实时金银价格**: 自动获取国际黄金/白银价格并换算为人民币
- 💱 **元/万元单位切换**: 各卡片独立单位切换，方便不同量级录入
- 📋 **详细资产分类**: 现金、贵金属、股票基金、投资房产、应收账款
- 💳 **债务扣除**: 日常开支、各类贷款、信用卡等
- 💾 **草稿自动保存**: 刷新页面后自动恢复填写内容
- 📄 **精美结果页**: 可视化展示计算结果与进度条
- 🖨️ **打印友好**: 结果页支持直接打印

## 文件结构

```
zakat-v2/
├── index.html      # 主计算页面
├── result.html     # 结果展示页面
└── README.md       # 说明文档
```

## 使用方法

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

数据存储使用 LocalStorage，完全在本地运行，保护隐私。
