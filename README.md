# 哈乃斐学派天课计算器

一个基于哈乃斐（Hanafi）学派教法的天课（Zakat）在线计算工具，纯静态网页，无需后端服务。

## 功能特性

- 🟡 **实时金银价格**：自动从国际市场获取黄金/白银价格并换算为人民币/克
- 💱 **元 / 万元 单位切换**：全表联动转换，适配不同量级的资产录入
- 📋 **详细资产分类**：现金、黄金、白银、股票基金、投资房产、应收账款
- 💳 **细化债务扣除**：个人欠款、房贷、车贷、信用卡、其他短期债务
- 💾 **草稿自动保存**：刷新或关闭页面后自动恢复未完成的填写内容
- 📄 **独立结果页**：跳转至专属结果页展示完整计算明细
- 🖨️ **打印友好**：结果页支持直接打印为清晰的黑白凭证

## 使用方法

直接用浏览器打开 `Zakat.html` 即可，无需安装任何依赖。

## 部署到 GitHub Pages

1. 将此仓库 Fork 或克隆至你的 GitHub 账号
2. 进入仓库 **Settings → Pages**
3. Source 选择 `main` 分支、`/(root)` 目录，保存
4. 稍等片刻，即可通过 `https://<你的用户名>.github.io/<仓库名>/Zakat.html` 访问

## 教法依据

- 学派：**哈乃斐（Hanafi）**
- 起征点（Nisab）标准：白银 **612.36 克**的当前市值
- 天课比例：**2.5%**
